Instruction how to add icons in SVG format

In order for a SVG file to display in correct size and color it has to meet 3 requirements:

  1. svg element has to have an id="icon" defined like this: <svg id="icon"
  2. svg element should have width and height removed: <svg width="24" height="24"
  3. paths should have defined fill="currentColor"
Example:
<svg id="icon" xmlns="http://www.w3.org/2000/svg" viewBox="0 0 5.62 11.2"><path fill="currentColor" d="M2.23,2.81H4.85L5.62,0H3M0,11.2H2.59l1.61-6H1.61Z"/></svg>

Instruction how to add icons from iconify

  1. Find an icon on Iconify page
  2. Select an icon and choose the folowing options:
    • Size: No size
    • Color: currentColor
    • Format: SVG
  3. Click "Download"
  4. Open the downloaded .svg file in text editor and add id="icon" in the beggining of svg element like this: <svg id="icon"
  5. Upload .svg file to umbraco media folder
