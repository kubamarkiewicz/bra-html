Instructions

Any SVG file can be used as an icon. Icons are stored in /assets/icons folder. Additional icons can be downloaded from Iconify.
SVG files specification
The SVG root element must have an id="icon" attribute.
In order to change the color of the icon, the SVG file must contain the fill="currentColor" attribute.
In order to change the size of the icon, the SVG file must not contain the width and height attributes.
