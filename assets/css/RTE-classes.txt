Styles that should be available in Rich Text Editor in "Format" dropdown menu:

Submenu "Paragraphs"
Long text >> html tag "p", CSS class "text-long"
Short text >> html tag "p", CSS class "text-short"
Fine print >> html tag "p", CSS class "text-fine-print"

Submenu "Headers"
H1: Display >> html tag "h1"
H2: Headline >> html tag "h2"
H3: Sub Headline >> html tag "h3"
H4: Paragraph Headline >> html tag "h4"
H5: Interactive Text >> html tag "h5"
H6: Form Field Headline >> html tag "h6"

Submenu "Text colors"
BRA Green >> CSS class "text-bra-green"
BRA Dark Green >> CSS class "text-bra-green-dark"

Submenu "Buttons"
Dark green with arrow >> CSS class "btn btn-secondary btn--arrow"


Stylesheet for RTE is located in "assets/css/rte.css"
